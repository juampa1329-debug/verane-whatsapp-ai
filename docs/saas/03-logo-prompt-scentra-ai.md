# Prompt de logo para Scentra +AI

## Prompt principal
Disena un logo premium para una plataforma SaaS llamada "Scentra +AI". La marca combina inteligencia artificial, automatizacion conversacional, CRM y comercio por WhatsApp para negocios de belleza, perfumeria, retail y servicios. El logo debe sentirse moderno, confiable, elegante y tecnologico, con un simbolo memorable basado en la letra "S", una onda de aroma/scent, una chispa de IA o una burbuja conversacional abstracta. Evitar robots genericos, cerebros literales y estilos demasiado infantiles. Usar una paleta sofisticada con verde esmeralda profundo, azul petroleo y un acento lima o dorado suave. Tipografia limpia, premium y legible, con personalidad. Crear version horizontal con isotipo + texto "Scentra +AI", version solo isotipo y version monocromatica. Fondo blanco, composicion centrada, alto contraste, vector logo, minimalista, escalable, apto para app icon, dashboard SaaS y favicon.

## Negative prompt
No usar mascotas caricaturescas, no usar iconos genericos de robot, no usar exceso de degradados, no usar imagenes 3D, no usar mockups, no usar fotografia, no usar fondos complejos, no usar texto ilegible, no usar estilos de criptomoneda, no usar colores neon excesivos.

## Direccion visual recomendada
1. Isotipo: "S" fluida con trazo que sugiera aroma + circuito sutil.
2. Color base: `#0f766e` o `#115e59`.
3. Acento: `#a3e635` o dorado suave `#d6b35a`.
4. Tipografia: sans serif premium con curvas humanas.
5. Sensacion: tecnologia calida, elegante y vendible para negocios reales.
